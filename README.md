Pratham Chikitse – First Aid Emergency Assistance App:
Pratham Chikitse is a bilingual Android-based first aid emergency assistance application developed during an industry internship at MindMatrix. The application is designed to provide quick and accessible first-aid guidance during emergency situations using a simple, user-friendly interface with English and Kannada language support.

The project focuses on improving emergency awareness and helping users access first-aid instructions instantly through mobile technology. 

## Features

* 🚑 Emergency first-aid guidance
* 🌐 Bilingual support (English & Kannada)
* 🔊 Text-to-Speech (TTS) voice assistance
* 🔍 Search functionality for emergency situations
* 📞 Emergency call integration
* 📋 RecyclerView-based emergency category listing
* 🎨 User-friendly and responsive UI
* 📶 Offline accessibility using JSON data
